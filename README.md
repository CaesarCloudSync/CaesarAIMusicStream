# CaesarAIMusicStream
A CaesarAIMusic song player

## Preview

![](media/preview-android.gif)

## How to run the app?

- Download or clone the repository.
- Install dependencies:
```bash
npm install
# --- or ---
yarn install
```
- Run the app:
```bash
npx react-native run-android
# --- or ---
npx react-native run-ios
```
